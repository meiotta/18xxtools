Getting Started

The map is your canvas. Click a tile in the left panel, then click a hex to place it. Right-click any hex for quick options — add a city, town, terrain, or resource icon directly.

Use File → Import Map to load an existing 18xx.games .rb file and reproduce a published map automatically.

Tiles & the Palette

Tiles are numbered track pieces from the 18xx catalog. Yellow → green → brown is the upgrade progression. Gray tiles are fixed and can't be upgraded in-game.

Use the Tile Manifest button (bottom-right of canvas) to browse all tiles for a specific game and drag them to the palette.

Shift+click repeats the last placed tile at the same rotation.

Static Hexes

Static hexes (gray, red, blue backgrounds) are pre-printed and don't change during a game — major cities, offboard areas, and ocean hexes.

Right-click a hex → Static Hex Wizard to build one visually. Gray hexes support pre-printed track, towns, cities, OO layouts, and phase-varying revenue.

Red hexes are offboard destinations with revenue that increases by phase.

Import & Export

Import Map (.rb) parses an 18xx.games Ruby source file and reproduces the full board — static hexes, terrain, borders, icons, and city names.

Export ZIP generates a map.rb file compatible with the 18xx.games engine, plus a JSON save of your session.

Save / Load JSON saves your editor state so you can resume later.

Map Configuration

The Config tab (right panel) controls hex orientation (flat-top or pointy-top), terrain costs, and map dimensions.

Use the Map Size controls to grow or shrink the grid. Shrinking warns you before deleting any defined hexes.

Keyboard & Mouse

Right-click any hex — quick add menu (city, town, terrain, icons)

Shift+click — repeat last tile at same rotation

Scroll wheel — zoom in/out

Middle-click drag or right-drag — pan the map

Erase button — clears a hex back to blank
